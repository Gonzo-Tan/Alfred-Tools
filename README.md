# 功能列表

## 1. 日期工具类

now: 获取当前时间，以各种格式展示 

ttd: 时间戳转日期

ddt: 日期转时间戳

## 2. 字符串工具类

base64en: base64编码

base64de: base64解码

md5: md5编码

urlen: url编码

urlde: url解码

- 以下是未支持功能

*strsw: 字符串格式切换（驼峰、下划线、全大写、全小写）*

*passgen: 密码生成*


## 3. json工具类

- 以下是未支持功能

*json2jb: json to java bean*

*json: json格式化展示*
